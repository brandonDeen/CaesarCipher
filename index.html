How to Encrypt

  1. Pick a plain text message you want to encrypt
  2. Select a cipher value
  3. Select Encrypt
  4. Click Go
  5. Your Encrypted message will appear in the section below the form

How to Decrypt

  1. Paste your decrypted message in the text area
  2. Select the cipher value used to encrypt the message
  3. Select Decrypt
  4. Click Go
  5. Your Decrypted message will appear in the section below the form
